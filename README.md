# ASMOS (Autonomous Self-Management OS)

ASMOSは、認知科学と生産性理論に基づき、ユーザーの認知バイアスを自動で補正し、行動を強制する自律型の自己管理アプリケーションです。
単なるメモツールではなく、パーキンソンの法則や計画錯誤といった心理的バイアスをシステムレベルで排除し、持続可能な高パフォーマンスを実現することを目的としています。

## 🚀 インストール方法

本プロジェクトはブラウザ上で動作する「ローカルファースト」な設計となっており、外部サーバーは不要です。

1. **リポジトリをクローンまたはダウンロードします**
   ```bash
   git clone https://github.com/Bar0919/Self-Optimizer.git
   cd Self-Optimizer
   ```

2. **依存関係をインストールします**
   ```bash
   npm install
   ```

3. **TypeScriptをビルドします**
   ```bash
   npm run build
   ```

4. **アプリケーションを起動します**
   ディレクトリ内の `index.html` を任意のWebブラウザで開いてください。

## 🛠 使い方

ASMOSは、以下の5つのフェーズ（PDCAサイクル）に沿って設計されています。

### 1. 📊 統合ダッシュボード (Fact)
一日の全体像を把握します。現在のバイタルスコア（精神的・肉体的エネルギー）や、バイタル推移グラフ、予定時間の合計、残タスク数などを一目で確認できます。

### 2. 🎯 目的管理 (Goal Management)
プロジェクト単位で中長期的な戦略をマインドマップ形式で管理します。
- 左側のサイドバーでプロジェクトを切り替え・作成・削除。
- Markdown形式で思考を整理し、右側のビジュアライザーで構造を俯瞰。

### 3. 💡 if-thenプランニング
「もし〜なら（If）、そのとき〜する（Then）」という行動トリガーを事前定義します。
- 習慣（Habit）、リスク回避（Risk）、業務（Work）のカテゴリ別にルールを整理。
- 意思決定の負荷を減らし、自動的な行動を促します。

### 4. ⏳ 実行管理 (Execution)
日々のタスクを3軸（ストック・計画・実行）のタイムラインで管理します。
- **⚡ 自動スケジューリング**: 本日ストックにあるタスクを、優先度と各スロットの空き時間、認知切り替えコストを考慮して自動配置します。
- **ホフスタッターの法則**: 見積時間はシステムが自動的に1.5倍に補正し、計画の破綻を防ぎます。
- **20%のスラック**: 各時間枠の20%は不確実性のために強制的に予約され、詰め込みすぎを防止します。

### 5. 🧠 内省とアクション (Review)
一日の終わりに高度な振り返りを行います。
- **予実分析**: 補正後の予定時間と実測時間の残差を確認し、見積精度を向上させます。
- **バイタル分析**: 一日の気分変動グラフを確認し、エネルギーの枯渇タイミングを特定。
- **5ステップ分析**: 事実 ➔ 状況 ➔ 論理（Why） ➔ インサイト ➔ 明日への判断。
